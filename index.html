My Magic Mirror

A futuristic smart mirror

My Magic Mirror is a mirror that displays metrics about the day. These metrics include weather, chance of rain, day of week, and other useful information while still functioning as an everyday mirror. The project is designed Mongodb, Express and Node. The app pulls data from a weather API and then manipulates the data to display easy to read information to the user.
